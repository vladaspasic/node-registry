    injection
    (
    • name
    • property
    • injectionName
    )

    Defined in lib/registry/container.js:224

    Define injections for a certain module. These injections will be applied when Modules are instantiated.

    container.register('user', User);
container.register('email', Email);
container.inject('email', 'user', 'user');

    or

    container.register('user', User);
container.inject('email', 'user', 'user');
container.register('email', Email);

    When performing injections, please notice that the injected value must be declared before the injection is declared and the target module is not alreay looked up.

    • name String
    • property String
    • injectionName String
