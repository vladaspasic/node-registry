createServer

(
  • app
)
Server

Defined in lib/registry/registry.js:133

Creates the Server Module. You will extend the Server module with new methods.

var Server = Registry.createServer(function(req, res) {
                                                res.write('ok');
                                            });
                                            
                                            Server.start();
                                            
var Server = Registry.createServer({
                                               port: 8080,
                                               listener: function(req, res) {
                                                  res.write('ok');
                                               }
                                            });
                                            
                                            Server.start();

If the server is already created, this method will raise an Error, as there can only be one instance of the Server registered in the Container.

registerInitializer

Defined in lib/registry/registry.js:62

Register an initializer function that will be invoked before the server is started.

Registry.registerInitializer({
                                               name: 'myInitializer',
                                               initializer: function(container, server, callback) {
                                                  // your logic
                                              });

When creating an initializer, name and initializer properties, are required.
